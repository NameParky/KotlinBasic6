# 💡 현재 프로젝트의 목표
>1. 여러개의 버튼을 단일로 관리할 수 있다.
>2. 입력값과 연산값을 바로바로 뷰에 출력할 수 있다.
>3. 연산버튼 사용간 충돌이 나지않게 플래그를 통해 버튼 제한을 줄 수 있다.
>4. 동일한 입력값으로 2개의 뷰에서 원하는 입력값만을 출력할 수 있다.
>5. 최종적으로 실제 계산기와 같이 사용할 수 있다.
